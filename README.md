# UGFNet
Estimate, Refine, and Reason: An Uncertainty-guided Geometric Foundation Network for RGB-D Indoor Semantic Segmentation

This repository provides the code and results for "Estimate, Refine, and Reason: An Uncertainty-guided Geometric Foundation Network for RGB-D Indoor Semantic Segmentation"

---

## Requirements

- Python 3.7+
- PyTorch 1.5+
- CUDA 10.2+
- TensorboardX 2.1+
- opencv-python
